# Windows 录音器 v2

Windows 系统声音和麦克风录音工具，支持 WAV/MP3 格式输出，明暗主题切换。

## 功能介绍

- 🎙️ **系统声音录制** - 录制电脑播放的音频（浏览器、音乐播放器等）
- 🎤 **麦克风录制** - 录制麦克风输入
- ⏸️ **暂停/继续** - 录音过程中可随时暂停和继续
- 📁 **WAV/MP3 格式** - 支持无损 WAV 和压缩 MP3 格式
- 🌓 **明暗主题** - 一键切换亮色和暗色主题
- 📍 **自动保存** - 录音文件自动保存到 Documents/WindowsRecorder 文件夹

## 使用方法

1. 下载并运行 `WindowsRecorder.exe`
2. 选择音频源（系统声音/麦克风）
3. 选择输出格式（WAV/MP3）
4. 点击"开始录音"按钮开始录制
5. 录音过程中可点击"暂停"按钮暂停，再次点击"继续"恢复
6. 录音完成后点击"停止录音"，文件将自动保存

## 系统要求

- Windows 10/11
- .NET 8 Runtime
- 音频播放设备

## 技术栈

- C# WPF
- .NET 8
- NAudio (音频录制)
- NAudio.Wasapi (系统音频捕获)

## 构建

```bash
dotnet build -c Release
```
